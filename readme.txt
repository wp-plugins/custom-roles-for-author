Allows to set users with custom role as post author.

== Description ==

This plugin allows you to set users with custom role as post, page or custom post type author.

In addition, this plugin allows you to set what default roles (even subscriber, contributor) shoud appear in author dropdown list.

Plugin developed with "**define('WP_DEBUG', true);**" on WordPress 3.3.1 => **No notices, errors, warnings**

== Installation ==

1. Download plugin
1. Extract all files to a folder
1. Upload folder with all files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set desired user roles to appear in authors drop down menu in Users -> Custom roles 4 author

== Frequently Asked Questions ==

= I don't see author meta box =

You must enable that box via "Screen options" tab in upper right hand corner.
